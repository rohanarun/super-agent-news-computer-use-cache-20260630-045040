How Super’s computer-use cache works

1. Run once with real computer use

Super operates a real browser or desktop environment, clicking, typing, scrolling, and navigating like a human assistant.

2. Save successful actions

When the task completes, Super stores the action trace as a reusable computer-use cache instead of throwing it away.

3. Replay for repeat runs

On the next run, Super replays the cached actions directly, skipping expensive exploration.

4. Heal when interfaces change

If a site changes, Super can fall back to live computer use and refresh the cache.
